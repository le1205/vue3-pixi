## Install

```sh
pnpm install vue3-pixi-ui
```

## Usage

using `vue3-pixi-ui` requires loading the module before using `vue3-pixi`

```ts
import { renderer } from 'vue3-pixi'
import UIRenderer from 'vue3-pixi-ui'

renderer.use(UIRenderer)
```
